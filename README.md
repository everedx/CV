# Web CV (GitHub Pages)

A clean, fast, single-page CV designed for GitHub Pages.

## Deploy on GitHub Pages

1. Create a repo named **<your-username>.github.io** (or use any repo and publish from `/docs` or branch).
2. Copy the contents of this folder into the repo root.
3. Commit + push.
4. In GitHub: **Settings â†’ Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` (root)

Your site will appear at: `https://<your-username>.github.io/`

## Edit your data

Open `index.html` and replace:
- Email / phone
- GitHub + LinkedIn links
- About summary
- Experience entries
- Projects
- Education + certifications

## PDF export

Click **PDF** (top right) to print / save as PDF. Print styles are included.
